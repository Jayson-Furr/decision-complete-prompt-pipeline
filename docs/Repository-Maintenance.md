# Repository Maintenance

Maintain these invariants:

```text
Human-readable first.
Stable IDs.
Stage authority.
Traceability.
Decision ledgers.
Controlled placeholders.
Review gates.
No unsupported fact fabrication.
No unauthorized side effects.
```

When adding new profiles, update `configuration-examples/`, `Pipeline-Configuration-Guide.md`, examples where useful, and run all checks.
