# Project Brief — Starfall Cartographer

## One-sentence concept

A cozy 2D Unity puzzle game where the player restores broken constellations by rotating, linking, and tuning star fragments across a hand-painted night sky.

## Target experience

The game should feel calm, tactile, luminous, and contemplative. It should be easy to learn, readable at a glance, and suitable for short sessions.

## Core loop

```text
1. Enter a constellation puzzle.
2. Inspect broken star fragments.
3. Rotate, link, and tune fragments.
4. Complete the constellation shape.
5. Receive a short lore card and unlock the next puzzle.
```

## Preferred toolchain

```text
Unity version: Unity 6000.4.11f1
Renderer: Universal Render Pipeline with 2D Renderer
UI: UI Toolkit
Input: Unity Input System
Target platforms: Windows and macOS first; WebGL later if feasible
```
