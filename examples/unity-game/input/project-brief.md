# Project Brief — Starfall Cartographer

## Product summary

A 2D constellation-restoration puzzle game where the player rotates star clusters, unlocks mythic sky maps, and builds a calm celestial atlas.

## Product type

```text
Unity game
```

## Design preferences

- Unity 6000.4.11f1
- UI Toolkit
- Universal Render Pipeline with 2D Renderer
- Unity Input System
- Desktop first: Windows and macOS
- Calm puzzle game tone
- No online services in MVP

## Initial feature candidates

- Main menu
- Constellation puzzle board
- Star cluster rotation
- Hint system
- Progression map
- Settings menu
- Accessibility options
- Local save data

## Known high-stakes or review-gated topics

- Third-party constellation reference art rights
- Public release authorization
- Store page publication
- Analytics intentionally excluded from MVP

## Validation focus

- Unity Editor import validation
- C# compile validation
- UI Toolkit import validation
- Play Mode smoke test
- Build validation

## Notes for Stage 1

This brief is intentionally incomplete. Stage 1 should produce a decision-complete design specification by interviewing for medium- and high-stakes decisions, creating controlled placeholders where needed, preserving review-gated issues, and avoiding unsupported facts.
