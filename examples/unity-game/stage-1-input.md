## Short description

A cozy 2D constellation puzzle game where players restore broken star maps by rotating star fragments, linking constellations, and unlocking gentle lore cards.

## Technical preferences

```text
Unity 6000.4.11f1
Universal Render Pipeline with 2D Renderer
UI Toolkit
Unity Input System
2D orthographic camera
Target desktop first: Windows and macOS
```

## Visual direction

```text
- deep blue and violet night sky;
- soft gold and pale cyan star glow;
- hand-painted nebula textures;
- elegant thin-line constellation diagrams;
- clean readable UI;
- no harsh neon cyberpunk style;
- no horror tone;
- no realistic astronomy simulation requirement.
```

## Gameplay notes

```text
- Player rotates star fragments.
- Player connects nearby star nodes.
- Incorrect links are shown gently, not punished harshly.
- Completion triggers a small constellation reveal animation.
- Each puzzle unlocks a short lore card.
```

## Accessibility preferences
