# Project Brief — CivicIssue Board

## One-sentence concept

A web application for small civic teams to collect, triage, assign, and track public infrastructure issues such as potholes, broken signs, and lighting problems.

## Target users

```text
- Residents who submit issues.
- Staff who triage submissions.
- Field workers who update issue status.
- Administrators who manage categories and users.
```

## Core workflows

```text
- submit issue;
- review issue queue;
- assign issue;
- update status;
- filter and search issues;
- view public issue map or list;
- export summary reports.
```

## High-stakes areas

```text
- authentication and authorization;
- personal data handling;
- location data;
- public visibility of reports;
- production deployment;
- external map provider terms;
- official municipal use claims.
```
