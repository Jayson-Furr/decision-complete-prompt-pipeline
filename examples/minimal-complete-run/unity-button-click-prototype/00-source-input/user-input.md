# User Input

Create a tiny Unity UI Toolkit prototype called **Button Click Prototype**.

The prototype should include:

```text
- one screen;
- a button labeled "Click";
- a visible counter starting at 0;
- each button click increments the counter by 1;
- no saving, accounts, analytics, payments, ads, networking, deployment, or publishing.
```

Preferred Unity defaults:

```text
Unity version: Unity 6000.4.11f1
UI framework: UI Toolkit
Input: pointer/mouse or gamepad submit through UI focus where Unity supports it
Target: local development prototype only
```
