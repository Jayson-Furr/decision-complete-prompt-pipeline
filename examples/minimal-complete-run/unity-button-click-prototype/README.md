This complete miniature run shows a Unity-specific four-stage pipeline for a tiny UI prototype.

Product name: Button Click Prototype

## Key lesson

The Stage 3 example can fabricate text-based Unity artifacts, such as C# scripts, UXML, USS, and Unity Editor creation packets. It must not claim actual `.unity` scenes, `.prefab` files, imports, compilation, tests, or builds succeeded unless those actions actually ran.
