# User Input

Create a tiny local-only Todo web app called **Tiny Tasks**.

The app should let a user:

```text
- add a todo item;
- mark a todo item complete or incomplete;
- delete a todo item;
- see a helpful empty state;
- use the app without accounts, backend services, analytics, payments, or deployment.
```

Preferred implementation style:

```text
React-style TypeScript components, simple CSS, in-memory state only, no persistence.
```
