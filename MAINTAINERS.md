# Maintainers

```text
Maintainer: @Jayson-Furr
Repository: Jayson-Furr/decision-complete-prompt-pipeline
Version: 1.0.0
```

The repository owner is responsible for reviewing changes to prompts, shared standards, schemas, validators, examples, release notes, and repository metadata.

## Maintainer responsibilities

- Preserve the four-stage authority model.
- Preserve artifact boundary markers.
- Preserve the standardized decision ledger schema.
- Preserve placeholder, review-gated, external-tool-required, and external-validation-required handling.
- Review profile-family additions for consistency with the shared standards.
- Keep release manifests and validation tools current.
- Avoid publishing unsupported claims, credentials, or authorization-sensitive content.
