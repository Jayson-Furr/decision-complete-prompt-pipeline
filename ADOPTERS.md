# Adopters Guide

This file is for people who want to copy, fork, adapt, or publish DecisionComplete Prompt Pipeline.

Before publishing your copy, review `MAINTAINERS.md`, `.github/CODEOWNERS`, `LICENSE`, `CITATION.cff`, `SECURITY.md`, `SUPPORT.md`, and `PUBLICATION-CHECKLIST.md`.

Safe customization areas include examples, product-type profiles, documentation, validators, and repository metadata. Use extra care when changing artifact boundary markers, stage authority rules, decision ledger schemas, ID namespace rules, controlled vocabularies, high-stakes handling, placeholder handling, review-gated handling, configuration priority rules, or handoff contracts.
